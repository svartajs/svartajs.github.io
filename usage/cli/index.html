Command line tool

Install

terminal
npm i -g @svarta/cli

Help

terminal
svarta helpsvarta [command] --help

Scaffold new project

terminal
svarta init --manager [npm,yarn,pnpm]

Same as create svarta-app

Run development server

terminal
svarta dev

Build server

terminal
svarta build

Deploy server

terminal
svarta deploy

Show version

terminal
svarta --version
