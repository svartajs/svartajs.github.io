# Command line tool

## Install

```bash
npm i -g @svarta/cli
```

## Help

```bash
svarta help
svarta [command] --help
```

## Scaffold new project

```bash
svarta init --manager [npm,yarn,pnpm]
```

Same as `create svarta-app`

## Run development server

```bash
svarta dev
```

## Build server

```bash
svarta build
```

## Deploy server

```bash
svarta deploy
```

## Show version

```bash
svarta --version
```
